Data Source & Integrity

Documents are retrieved from the National Archives NextGen Catalog API v2 and stored locally in documents/founding/. Each file includes retrieval metadata, NAID, source URL, and verification timestamps.

Documents are automatically refreshed via scheduled process and can be manually verified at catalog.archives.gov.

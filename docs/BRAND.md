# Evident Brand & Design System

## Brand Overview

- **Legal Entity:** Evident Technologies, LLC
- **DBA (Doing Business As):** EVIDENT
- **Primary Tagline:** Cut away all the extra. Get to the truth.
- **Brand Voice:** Evident.
- **Mission:** Born from lived experience. Evidence processing shouldn't require

- **Core Values:** Truth before persuasion, integrity before convenience,
  restraint before expression, due process before outcomes
- **Primary Color:** `--brand: #2F5D9F;` (Evident Blue)
- **Accent Color:** `--accent: #F7B32B;` (Gold)

- **Ink:** `--ink: #0B0D10;` (Near Black)
- **Muted:** `--muted: #6B7280;` (Muted Gray)
- **Pattern:** Subtle grid/pattern backgrounds for structure
- **Typography:** Modern, accessible sans-serif (see tokens.css)
- **Logo:** Scales of Justice, geometric, blue/gold

## Design Tokens (see assets/css/tokens.css)

- All colors, radii, shadows, spacing, and typography are defined as CSS
  variables in `assets/css/tokens.css`.
- Use only tokens for all UI elements.

## UI System

- **Buttons:** Unified `.btn` system (primary, secondary, ghost), pill radius,
  accessible focus, no gradients
- **Layout:** `.container`, `.grid`, `.card`, `.panel`, `.bg-grid` for
  responsive, accessible layouts
- **Patterns:** Subtle SVG/CSS grid backgrounds for hero, footer, and cards
- **Accessibility:** All interactive elements have visible focus, sufficient
  contrast, and semantic markup

## Usage

- Never use Evident or legacy branding
- All UI, docs, and assets must reference Evident
- Forensic, accessible, and trustworthy tone in all content
